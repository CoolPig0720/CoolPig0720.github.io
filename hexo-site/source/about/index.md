## 📌 关于本站

本站基于 [Hexo](https://hexo.io/) + [Butterfly](https://butterfly.js.org/) 主题搭建，并托管在 GitHub Pages 上。GitHub Pages 是一个免费的服务，允许你通过 GitHub 仓库托管个人网站，每次提交代码后网站会自动更新。

Butterfly 是一款现代化的 Hexo 主题，具有以下特点：
- 简洁美观的设计
- 优秀的响应式布局
- 丰富的功能支持（搜索、评论、目录等）
- 高性能优化
- 完善的中文支持

## ✨ 支持的功能

本站支持以下实用工具：
- [MathJax](https://www.mathjax.org/) - 用于数学公式
- [Mermaid](https://mermaid.js.org/) - 用于绘制图表
- 代码高亮与语法展示
- 字数统计与阅读时间估算
