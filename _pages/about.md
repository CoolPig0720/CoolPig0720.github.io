# 欢迎来到我的个人网站

你好！我是 CoolPig，这是我的个人网站。

这里是我分享想法、记录学习过程的地方。你可以在这里找到我的博客文章、作品集，以及其他我想分享的内容。

## 关于本站

本站基于 [Academic Pages](https://github.com/academicpages/academicpages.github.io) 模板搭建，并托管在 GitHub Pages 上。GitHub Pages 是一个免费的服务，允许你通过 GitHub 仓库托管个人网站，每次提交代码后网站会自动更新。

这个模板最初由 Michael Rose 创建的 [Minimal Mistakes Jekyll Theme](https://mmistakes.github.io/minimal-mistakes/) 改编而来，特别适合展示学术内容，如论文、报告、教学内容、作品集、博客文章等。当然，它也非常适合用来展示个人专业网站！

## 支持的功能

本站支持以下实用工具：
- [MathJax](https://www.mathjax.org/) - 用于数学公式
- [Mermaid](https://mermaid.js.org/) - 用于绘制图表
- [Plotly](https://plotly.com/javascript/) - 用于数据可视化
