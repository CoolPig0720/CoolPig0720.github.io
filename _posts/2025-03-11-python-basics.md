# Python 基础入门

## 1. 变量与数据类型

Python 支持多种数据类型：

```python
name = "CoolPig"      # 字符串
age = 25              # 整数
height = 1.75         # 浮点数
is_student = True     # 布尔值
```

## 2. 列表操作

```python
fruits = ["苹果", "香蕉", "橙子"]
fruits.append("葡萄")
print(fruits[0])  # 输出：苹果
```

## 3. 总结

Python 语法简洁，非常适合初学者入门。
